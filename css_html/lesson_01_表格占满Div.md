需求如下： 左边一个DIV， 右边一个Div。 点击左边的按钮显现右边的DIV。 两边的DIV都是可以关闭。
```
<html>
<head>
<style type="text/css">
.b{
  height:300px;
  width:300px;
}
table{
  width:100%;
  height:100%;
}
.a{
  width:100%;
  height:100%;
}
</style>
</head>

<body>
<div style="background-color:yellow;" class="b">
<table><tbody><tr>
<td><div style="background-color:red;" class="a">a</div></td>
<td><div style="background-color:green;" class="a">b</div></td>
</tr></tbody></table>

</div>
</body>

</html>

```
